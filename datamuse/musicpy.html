Programming with Music.py

Here's the basic syntax for melodies:
+ Create a rhythm. b means a beat, and a - means sustain. A half note would be b - + Create a melody. The letter is the note name, and the number is the octave. Middle C is C4 + Apply the rhythm to the melody. Each b will correspond to the next note in the melody. + Put that chord/rhythm combo in a play() function, along with the tempo and the instrument you want to use. As we go along, we'll include more instruments and more complex versions.

We've connected this tool to a database of convenience store sales provided by MurphyUSA. Hundreds of transactions are saved here, with the time of purchase, cost of the item, name of the item, and the category.

If we assign each minute of the day to a beat, and give the song a super fast tempo, we can quickly hear every entry over the course of the day. We can create a separate track for each category type, each with a giant lists of "b", "-", and "+". Then we add on the note values, and assign each track its own sound.
